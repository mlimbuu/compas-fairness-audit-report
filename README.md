# Auditing for Bias: Fairness in Classifier Models and Trade-offs using COMPAS

This repository archives a 2022 empirical research report on fairness trade-offs in classifier models using the COMPAS recidivism dataset.

The project examined whether removing protected attributes such as race reduces measured bias, compared simple classifiers with fairness-aware classifiers, and analyzed trade-offs between false positive parity, demographic calibration, and accuracy.

Note: The original experiment code is no longer available (lost). Hence, this repository is maintained as an archived report artifact for visibility and documentation of the analysis.
